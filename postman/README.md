# Postman Collection

This folder contains the Postman collection used to test the endpoints of the LovingPets project.

## Files
- `LovingPets.postman_collection.json`: Postman collection with the available API requests.

## Usage
1. Open Postman.
2. Click on **Import**.
3. Select the file `LovingPets.postman_collection.json`.
4. Start the microservice you want to test.
5. Use the collection to test the available endpoints.
